SkyPilot has announced a zero-egress storage feature for AI workloads on Hugging Face as of October 2023. This new capability allows users to run AI workloads on any cloud platform while storing data on Hugging Face without incurring data transfer fees. This development is particularly beneficial for organizations looking to optimize costs associated with data storage and transfer, enhancing flexibility in cloud resource utilization. The integration of zero-egress storage aligns with Hugging Face's ongoing efforts to streamline AI model deployment and management, following their recent updates that increased engagement by 50% with support for 1 billion parameters. For more details, see the [Hugging Face Blog](https://huggingface.co/blog/skypilot-hf-storage).
