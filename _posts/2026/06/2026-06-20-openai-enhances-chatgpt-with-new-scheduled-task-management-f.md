OpenAI has upgraded ChatGPT with a new scheduled task management feature, which consolidates all active tasks into a dedicated sidebar page. This feature allows users to manage their tasks more effectively, as it includes a functionality where research tasks can search the web and connected applications, sending alerts only when relevant changes occur. The previous Pulse feature is being retired as part of this update. This enhancement aims to position ChatGPT more firmly as a personal assistant tool, streamlining user interactions with the AI. The changes follow OpenAI's ongoing efforts to improve user experience and task management capabilities in their AI products, as noted in recent developments covered by [The Decoder](https://the-decoder.com/chatgpt-keeps-creeping-toward-becoming-your-ai-personal-assistant-with-new-scheduled-task-controls/).
