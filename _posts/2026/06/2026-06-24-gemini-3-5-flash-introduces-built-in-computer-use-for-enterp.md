On June 24, 2026, Google DeepMind released Gemini 3.5 Flash, which now includes computer use as a built-in tool, enhancing its capabilities for enterprise automation tasks. This feature was previously available only in the standalone Gemini 2.5 model. Mateo Quiros, Product Manager at Google DeepMind, stated that this version offers the 'best performance yet' for agentic computer use tasks, including continuous software testing. The platform also addresses accessibility issues through dedicated audits. Gemini 3.5 Flash utilizes targeted adversarial training for its computer use capabilities and requires explicit user confirmation for sensitive actions, automatically halting tasks if an indirect prompt injection is detected. This release follows earlier developments in the Gemini series, including the Gemini API and the Gemini Enterprise Agent Platform, which continue to support developers and enterprises respectively. For more details, visit the [Google DeepMind Blog](https://deepmind.google/blog/introducing-computer-use-in-gemini-3-5-flash/).
