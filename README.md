# Turing Wire

> The pulse of AI — news, research, and markets.

Turing Wire is an automated news aggregator that ingests RSS/Atom feeds and APIs from leading AI labs, hardware vendors, research organizations, and science journals. It publishes near-realtime summarized posts organized into three feeds: **News**, **Research**, and **AIStocks**.

Live site: [turingwire.com](https://turingwire.com)

---

## How it works

1. **Ingest** (hourly at :17) — `scripts/fetch_feeds.py` pulls from sources in `feeds/*.yml`, deduplicates, classifies via OpenAI gpt-4o-mini, and summarizes new articles.
2. **Stocks** (every 15 min, US market hours) — `scripts/fetch_stocks.py` pulls quotes from Finnhub and updates `_data/stocks_snapshot.json`. `scripts/compute_ai_index.py` maintains the AI Index history.
3. **Build** — Jekyll compiles the site, Pagefind indexes it, and the result is pushed to the `gh-pages` branch where GitHub Pages serves it.

### Branch layout

| Branch | Purpose |
|--------|---------|
| `main` | Source: scripts, feeds config, posts, Jekyll source |
| `gh-pages` | Built site only — auto-generated by CI, never edited by hand |

### Key files

```
feeds/
  news_sources.yml        # news feed source list
  research_sources.yml    # research feed source list
  company_aliases.yml     # company name normalization
scripts/
  fetch_feeds.py          # HTTP fetch + ETag caching
  deduplicate.py          # URL/title/body dedup
  classify_article.py     # LLM classification + impact rating
  summarize_news.py       # LLM news summary (250–350 words)
  summarize_research.py   # LLM research summary (450–550 words)
  tag_company.py          # company tagging (alias + LLM fallback)
  fetch_stocks.py         # Finnhub quote fetcher
  compute_ai_index.py     # equal-weight AI Index calculation
_data/
  seen_articles.json      # GUID cache (committed pipeline state)
  stocks_snapshot.json    # latest quotes (committed)
  ai_index_history.json   # index time-series (committed)
  tickers.yml             # AI Index ticker universe
```

## Running locally

### Prerequisites
- Ruby ≥ 3.1, Bundler
- Python ≥ 3.11
- `OPENAI_API_KEY` and `FINNHUB_API_KEY` in your environment

### Site

```bash
bundle install
bundle exec jekyll serve --livereload
```

### Pipeline (dry run — no LLM calls)

```bash
pip install -r requirements.txt
python scripts/fetch_feeds.py --dry-run
```

### Full pipeline run

```bash
export OPENAI_API_KEY=sk-...
export FINNHUB_API_KEY=...
python scripts/fetch_feeds.py
python scripts/deduplicate.py
python scripts/classify_article.py
python scripts/summarize_news.py
python scripts/summarize_research.py
python scripts/tag_company.py
```
