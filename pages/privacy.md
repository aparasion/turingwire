## Overview

Turing Wire is a static website hosted on GitHub Pages. It does not collect, store, or process personal data.

## What we collect

**Nothing.** This site has no user accounts, no login, no comments, no forms that collect personal information, and no analytics that track individuals.

## GitHub Pages

This site is hosted by GitHub (Microsoft). GitHub may collect server logs including IP addresses as part of their infrastructure. See [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) for details.

## External services

- **Google Fonts** — Fonts are loaded from Google's CDN. Google may log requests. To avoid this, use a browser extension that blocks external font requests.
- **Finnhub** — Stock data is fetched server-side by GitHub Actions. No client-side requests are made to Finnhub from your browser.
- **CDN resources** — Tailwind CSS and Apache ECharts are loaded from CDNs (cdn.tailwindcss.com, cdn.jsdelivr.net). These CDNs may collect standard server logs.

## Cookies and local storage

This site uses `localStorage` to remember your dark/light mode preference. No cookies are set. `localStorage` data never leaves your browser.

## Changes

This policy may be updated. The last-updated date at the top reflects the most recent change.
